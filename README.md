# Speech Processing Lab Assignment 8

Title
Spectral Analysis of Speech using DCT and MFCC Features

Author
Yeseswini

Description
This project analyzes speech signals using:
- Discrete Fourier Transform (DFT)
- Discrete Cosine Transform (DCT)
- Mel Frequency Cepstral Coefficients (MFCC)

Features
- Energy compaction comparison (DFT vs DCT)
- Histogram visualization
- MFCC feature extraction
- Time-varying speech analysis

Tools Used
- Python
- Librosa
- NumPy
- Matplotlib

Observations
- DCT theoretically provides better energy compaction
- MFCC captures speech dynamics effectively
- Silent frames may cause NaN energy values

## How to Run
1. Upload speech.wav
2. Run the notebook
3. View plots and results

## Output
- DFT vs DCT histograms
- MFCC heatmap
- Frame comparison plots
